# Me-OW
 > "Atirei o pau no gato-to..."
 
 O retorno de Dona Chica
 
 `Enter` para iniciar
 
 `W`/`A`/`S`/`D` ou `↑`/`←`/`↓`/`→` para controlar o personagem
 
 `Esc` para pausar
 
 ## Como baixar
 Na direita, aperte `Releases` e baixe o `.zip` do seu sistema operacional mais recente.
